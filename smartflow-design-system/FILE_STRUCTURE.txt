===============================================
HOW TO USE THESE FILES
===============================================

1. READ FIRST:
   - README.md (overview)
   - SMARTFLOW_SETUP_GUIDE.md (setup steps)

2. COPY THEME:
   cp styles/smartflow-theme.css your-app/client/src/index.css

3. COPY COMPONENTS YOU NEED:
   cp components/CircuitBackground.tsx your-app/client/src/components/
   cp components/ScrollToTop.tsx your-app/client/src/components/
   cp components/Navigation-Example.tsx your-app/client/src/components/Navigation.tsx

4. USE EXAMPLES AS TEMPLATES:
   - Copy example files to your components folder
   - Customize content, icons, and links
   - Combine patterns for your pages
