===============================================
QUICK START (5 MINUTES)
===============================================

1. Extract this package to your workspace

2. Copy theme CSS:
   cp smartflow-design-system/styles/smartflow-theme.css your-project/client/src/index.css

3. Copy components you need:
   cp smartflow-design-system/components/*.tsx your-project/client/src/components/

4. Add fonts to your index.html (see SMARTFLOW_SETUP_GUIDE.md)

5. Configure Tailwind (see SMARTFLOW_SETUP_GUIDE.md)

6. Start building with the example patterns!
